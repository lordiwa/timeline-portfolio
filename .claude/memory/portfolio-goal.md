Construir un portafolio personal para Rafael Matovelle ([[user-profile]]), decorado con pixel art generado vía pixelforge-mcp y post-procesado con Adobe MCP.

**Why:** Rafael tiene background de gameplay programmer + AR/VR, así que el pixel art encaja con su narrativa profesional, no es decorativo arbitrario. Además quiere usar este proyecto como vehículo para ejercitar los MCPs de arte.

**How to apply:**
- Las decisiones de diseño (paleta, vibe, secciones) deben apoyarse en su perfil real, no en un portfolio genérico de SE.
- Stack ya andamiado: Vue 3 + Vite + Phaser (Phaser opcional, ver [[stack-decisions]] cuando se cree).
- Workflow: planner → artist-creator (genera) → artist-editor (post-procesa) → frontend-dev (integra) → qa (verifica).
- Idioma, secciones y tono del portafolio están pendientes de decidir con el usuario (mayo 2026).
